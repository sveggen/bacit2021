# transform it

## Nettsted for bachelorgruppen "transform it" ved UiA

---

## Hvordan laste ned

**1. Installer repository**  
```
gatsby new bacit2021 https://github.com/sveggen/bacit2021
```

**2. Gå inn i mappa**  
```
cd bacit2021
```

**3. Starter en lokal server på port 8000**  
```
gatsby develop
```
